# Security Policy

## Intended Use

This repository contains security research tools designed for:

- **Authorized penetration testing** engagements
- **CTF (Capture The Flag)** competitions
- **Security research** and education
- **Defensive analysis** — understanding attack techniques to build better defenses

## Responsible Use

Using these tools against systems without **explicit written authorization** from the system owner is **illegal** and **unethical**. The author does not condone or support any unauthorized or malicious use.

## Legal References

- [Computer Fraud and Abuse Act (CFAA)](https://www.law.cornell.edu/uscode/text/18/1030) — United States
- [Computer Misuse Act 1990](https://www.legislation.gov.uk/ukpga/1990/18/contents) — United Kingdom
- [Convention on Cybercrime (Budapest Convention)](https://www.coe.int/en/web/cybercrime/the-budapest-convention) — International
